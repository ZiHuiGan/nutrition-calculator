# 📱 手机使用指南 - 添加到主屏幕

## 🌐 部署链接

**应用已成功部署**：
- 🔗 **访问地址**：https://nutrition-calculator-pwa.vercel.app
- 📱 **PWA 支持**：已启用，可添加到主屏幕
- ✨ **移动端优化**：已针对 iPhone 15 Pro Max 及主流手机优化

## 🍎 iOS (iPhone/iPad) - 添加到主屏幕

### 方法 1: Safari 浏览器（推荐）

1. **打开应用链接**
   - 在 iPhone Safari 浏览器中打开：https://nutrition-calculator-pwa.vercel.app

2. **点击分享按钮**
   - 点击 Safari 底部的**分享按钮**（方框带向上箭头图标）

3. **选择"添加到主屏幕"**
   - 在分享菜单中，向下滑动找到 **"添加到主屏幕"** 选项
   - 点击它

4. **自定义名称（可选）**
   - 系统会显示应用图标和名称（默认："营养计算器"）
   - 可以修改名称，点击右上角 **"添加"** 按钮

5. **完成**
   - 应用图标会出现在主屏幕上
   - 点击图标即可像原生 App 一样使用

### 方法 2: Chrome 浏览器（iOS）

1. 在 Chrome 中打开部署链接
2. 点击右上角 **菜单**（三个点）
3. 选择 **"添加到主屏幕"** 或 **"分享"** → **"添加到主屏幕"**
4. 点击 **"添加"**

## 🤖 Android - 添加到主屏幕

### Chrome 浏览器（推荐）

1. **打开应用链接**
   - 在 Chrome 浏览器中打开：https://nutrition-calculator-pwa.vercel.app

2. **点击菜单**
   - 点击右上角 **菜单**（三个点图标）

3. **选择"添加到主屏幕"**
   - 在菜单中选择 **"添加到主屏幕"** 或 **"安装应用"**
   - 如果显示 "安装应用"，说明 PWA 支持已启用，直接安装即可

4. **确认添加**
   - 系统会显示应用名称和图标
   - 点击 **"添加"** 或 **"安装"**

5. **完成**
   - 应用图标会出现在主屏幕或应用抽屉中
   - 点击图标即可使用

### 其他浏览器

**Firefox**:
- 菜单 → "页面" → "添加到主屏幕"

**Samsung Internet**:
- 菜单 → "添加到主屏幕"

## ✨ 移动端优化特性

### iPhone 15 Pro Max 优化

- ✅ **更大的触摸目标**：所有按钮和输入框至少 48-52px 高度
- ✅ **优化的间距**：更舒适的布局和间距
- ✅ **防止自动缩放**：输入框字体 16px，防止 iOS 自动缩放
- ✅ **流畅的滚动**：优化的滚动体验
- ✅ **安全区域适配**：适配 iPhone 刘海和底部安全区域

### 交互优化

- ✅ **触摸反馈**：点击时有视觉反馈
- ✅ **禁用双击缩放**：防止误触缩放
- ✅ **优化的选择框**：自定义样式的下拉菜单
- ✅ **更大的卡片**：更易于点击和阅读

## ✨ 使用体验

添加到主屏幕后：

- ✅ **全屏体验**：打开后没有浏览器地址栏，像原生 App
- ✅ **离线使用**：部分功能可以离线使用（取决于缓存）
- ✅ **快速访问**：从主屏幕一键打开，无需打开浏览器
- ✅ **流畅交互**：优化的触摸体验，响应迅速
- ✅ **通知支持**：如果未来添加通知功能，可以接收推送

## 🔄 更新应用

应用会**自动更新**：
- 当你推送新代码到 GitHub
- Vercel 自动重新部署
- 用户刷新页面即可看到更新（无需重新添加到主屏幕）

## 📝 注意事项

1. **首次打开可能较慢**：首次加载时需要下载资源
2. **需要网络连接**：虽然支持离线缓存，但计算功能需要加载数据
3. **浏览器兼容性**：
   - iOS 12+ 支持添加到主屏幕
   - Android 5.0+ 支持 PWA
   - 推荐使用最新版本的浏览器
4. **最佳体验**：推荐使用 Safari (iOS) 或 Chrome (Android)

## 🆘 常见问题

**Q: 找不到"添加到主屏幕"选项？**
- iOS Safari：确保使用的是 Safari，不是其他浏览器
- Android：确保使用 Chrome 或其他支持的浏览器

**Q: 添加到主屏幕后还是显示浏览器？**
- iOS：正常现象，iOS 的 PWA 会在 Safari 中运行，但不显示地址栏
- Android：Chrome 的 PWA 支持更完整，会以独立应用形式运行

**Q: 图标显示不正确？**
- 清除浏览器缓存后重试
- 确保 `icons/apple-touch-icon.png` 文件存在且大小为 180x180

**Q: 页面交互不流畅？**
- 已针对 iPhone 15 Pro Max 优化，确保使用最新版本
- 清除浏览器缓存后重试
- 检查网络连接
